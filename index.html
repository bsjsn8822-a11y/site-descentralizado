Hospedagem P2P ativa

Este site vive na Blockchain

Você está acessando uma aplicação estática totalmente descentralizada, imutável e distribuída globalmente via IPFS.

Status da Infraestrutura: 100% Descentralizado

1

O Conceito: Adeus Servidores Centrais

O código deste site não está guardado em um servidor convencional de empresas de tecnologia (como AWS ou Google). Em vez disso, ele foi carregado no IPFS (InterPlanetary File System), uma rede ponto-a-ponto (P2P) baseada em conceitos de blockchain onde arquivos são localizados pelo seu conteúdo criptográfico (Hash) e não pelo local físico.

2

A Camada de Publicação (Fleek)

Para automatizar o envio do código para a rede descentralizada de graça, foi utilizada a plataforma Fleek conectada a um repositório Git. Sempre que uma atualização de código acontece, a rede empurra automaticamente os novos arquivos para o IPFS, gerando uma URL descentralizada e imutável que se parece com isso:

ipfs://QmXoypizjW3WknFiJnKLwHCnL72vedxjQkDDP1mXWo6uco
3

A Ponte Web2 com a Cloudflare

Como os navegadores tradicionais (Chrome, Safari) não conseguem ler o protocolo ipfs:// por padrão, configuramos o domínio comum na Cloudflare para agir como um Gateway reverso. No painel de controle da Cloudflare, inserimos os registros DNS apontando para o Fleek:

Tipo: CNAME  |  Nome: @ (ou subdomínio)  |  Destino: domains.onfleek.app

Ao ativar a Nuvem Laranja (Proxy) na Cloudflare, ela cria uma cópia (cache) das páginas espalhada em mais de 300 cidades do mundo inteiro. Isso faz com que a hospedagem seja totalmente gratuita e o carregamento aconteça em milissegundos.
